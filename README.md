## Usage Guide

1. Clone the repo:

```bash
git clone https://github.com/Keobkeig/Foo-Food-Fighters
cd Foo-Food-Fighters
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```
